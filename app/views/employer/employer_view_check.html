View someone’s DBS result online

You can use this service to view someone’s DBS Online Result

What you will need

To use this service you will need:

  • the applicant's one-time share code
  • the last six digits of the applicant's certificate number
{% from "govuk/components/inset-text/macro.njk" import govukInsetText %} {{ govukInsetText({ text: "You can only use the share code once and you must use it within 21 days." }) }}

If you need to view the result again you will have to ask the applicant for another share code.

We will let the applicant know when you have viewed their result.

If you need proof that you have viewed a DBS Online Result

This service will also allow you to download a PDF receipt as proof you have viewed an applicant's Online Result.

The receipt includes:

  • their name
  • the post applied for - for example, teacher
  • their DBS certificate number and date of issue
  • if the check is standard or enhanced and the workforce it is for
  • if any barred lists were checked
  • the date you viewed the online result

The receipt contains information from the applicant's DBS certificate. If you download a copy, you must follow your organisation's policy on the storage, handling, use, retention and disposal of DBS certificate information.
