Dear Bill Carr,

Application reference number: E0123456789

You can share your DBS Online Result with someone else, like an employer. Your DBS online result contains sensitive personal information. You should be sure who is viewing your result before you share it.

Sharing your result

To share your result with someone else they will need:

  • a share code
  • the last six digits of your certificate number: 176560
  • The result link

You can forward this email to them but you must send the share code separately. For example, by phone. You can only use the share code once and it must be used within 21 days. You can get a new share code if your result needs to be viewed again.

Viewing your result

To view your result, the person with the share code will need to:

  • go to the DBS online result service
  • enter the share code and the last six digits of your certificate number into the right sections

We will let you know when your result has been viewed.

Cancel a share code

You can cancel a share code if it has not expired or been used. To do this you must sign in again and select ‘Cancel’ for the code on the ‘Manage share codes’ tab.

Contact DBS
