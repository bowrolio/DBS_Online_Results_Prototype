You can share your DBS Online Result with someone else, like an employer. Your DBS Online Result contains highly sensitive personal information. You should be sure about who you are allowing to view your DBS Online Result before you share it with them.

Use a share code

You need to give them:

  • a share code
  • the last six digits of your certificate number ******334455
  • the web link: https://dbs-or-alpha.herokuapp.com/employer_view_check

You can forward this email to them but it is important to send the share code separately. For example, by phone. You can only use the share code once and it must be used within 21 days.

They need to:

  • select the web link: https://dbs-or-alpha.herokuapp.com/employer_view_check
  • copy and paste the last six digits of your certificate number and share code into the right sections

We let applicants know when their DBS Online Result has been viewed. A share code can only be used once and must be used within 21 days. Applicants can get a new share code if their DBS Online Result needs to be viewed again.

Cancel a share code

You can cancel a share code if it hasn’t expired or been used. Log back in and select ‘Cancel’ for the code on the ‘Manage Share Codes’ tab.
